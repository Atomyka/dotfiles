"Default color scheme"
colorscheme wombat256 

"Display number lines"
set number

"Automatically load the file if it is modified"
set autoread

"no backup file - just use git"
set nobackup
set noswapfile

"Always show status line"
set laststatus=2

"Indent settings"
set tabstop=4
set shiftwidth=4
set smartindent

"Set the ruler to 80 columns"
set colorcolumn=80

"Ignore case in search patterns"
set ignorecase

"Keep at least 3 lines above/below"
set scrolloff=3

"Show immediately where the so-far typed pattern matches."
set incsearch

"Control-s to save"
:nmap <c-s> :w<CR>
:imap <c-s> <Esc>:w<CR>a
:imap <c-s> <Esc><c-s>

"When we hit enter, a new line is created on top/bottom without moving"
"the cursor."
nmap <S-Enter> O<Esc>
nmap <CR> o<Esc>

"This shifts the lines up or down"
nnoremap <S-j> :m .+1<CR>==
nnoremap <S-k> :m .-2<CR>==
inoremap <S-j> <Esc>:m .+1<CR>==gi
inoremap <S-k> <Esc>:m .-2<CR>==gi
vnoremap <S-j> :m '>+1<CR>gv=gv
vnoremap <S-k> :m '<-2<CR>gv=gv

"Remap ; to : in normal mode - More efficient!"
nnoremap ; :

"find using f"
nmap f /
