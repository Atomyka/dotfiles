"Default color scheme"
colorscheme wombat256 

"Display number lines"
set number

"Indent settings"
set tabstop=4
set shiftwidth=4
set smartindent

"Set the ruler to 80 columns"
set colorcolumn=80

"Ignore case in search patterns"
set ignorecase

"Keep at least 3 lines above/below"
set scrolloff=3

"Show immediately where the so-far typed pattern matches."
set incsearch

"Control-s to save"
:nmap <c-s> :w<CR>
:imap <c-s> <Esc>:w<CR>a
:imap <c-s> <Esc><c-s>

"When we hit enter, a new line is created on top/bottom without moving"
"the cursor."
nmap <S-Enter> O<Esc>
nmap <CR> o<Esc>

"This shifts the lines up or down"
nnoremap <A-j> :m .+1<CR>==
nnoremap <A-k> :m .-2<CR>==
inoremap <A-j> <Esc>:m .+1<CR>==gi
inoremap <A-k> <Esc>:m .-2<CR>==gi
vnoremap <A-j> :m '>+1<CR>gv=gv
vnoremap <A-k> :m '<-2<CR>gv=gv

"Remap ; to : in normal mode - More efficient!"
nnoremap ; :
